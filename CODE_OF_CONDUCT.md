# Code of Conduct

We aim to keep this project welcoming and harassment-free for everyone.

## Our standards
- Be respectful and professional.
- Assume good intent; ask clarifying questions.
- No harassment, hate speech, or discriminatory behavior.
- No doxxing or sharing private information.

## Enforcement
If you see an issue, report it via the channel in `SECURITY.md` or open a GitHub issue if appropriate.
Project maintainers may remove content and/or restrict participation for violations.
