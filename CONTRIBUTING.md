# Contributing

Thanks for helping improve KAS Racing.

## Quickstart
```bash
pnpm install
pnpm lint
pnpm test
pnpm build
pnpm dev
```

## Guidelines
- Keep changes focused and small when possible.
- Do not commit secrets (private keys, seeds, API keys, `.env`).
- Prefer TypeScript and keep strict types.
- Run `pnpm lint && pnpm test && pnpm build` before opening a PR.

## Pull requests
- Describe what changed and why.
- Include steps to test locally.
